GIT KRAKEN
La interfaz gráfica multiplataforma para git

¿Qué es GitKraken?


Es una potente y elegante interfaz gráfica multiplataforma para git desarrollada con Electron. De forma muy sencilla podemos llevar el completo seguimiento de nuestros repositorios, ver ramas, tags, crear nuevos, todo el historial de nuestro trabajo, commits etcétera.



Beneficios


  • -Facilita el gestionamiento del repositorio.
  • -Visualiza mejor los branch y sus acciones.
  • -Realiza commits, tags, etc rapidamente.
  • -Crea o Clona un repositorio mediante un formulario.
  • -Vincula facilmente tus cuentas de Github, Gitlab, entre otros.

Opinion Personal


Al encontrar Gitkraken y usarlo me doy cuenta de que las tareas basicas que son el crear repositorios, conectarlos con github, subir y bajar archivos, etc, se vuelve tan sencillo ya que solo tenemos que dar unos cuantos clicks para realizar dicha tarea.

Lo que más me gusto de esta interfaz es la visualizacion de las ramas y el historial de commits, que generalmente en la consola se ve muy "resumido".

Sinceramente te invito a usarlo ya que asi obtendras mas rapidez y control sobre tus proyectos.
